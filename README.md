# icloud-mail

Force light reading pane for iCloud Mail so text is readable on dark mode.

Set it as a custom `.css` style using [Stylus](https://add0n.com/stylus.html) extension for your browser.

Check it out on [UserStyles.world](https://userstyles.world/style/26172/icloud-mail-force-light-reading-pane) repository.
